# 🤖 Simple AI Agent with Memory & Tools (n8n)

This is a basic AI Agent built using n8n that can:
- Respond to chat messages
- Use a chat model (Google Gemini)
- Store conversation context using memory
- Fetch information from external tools (Wikipedia)

## ⚙️ Workflow Overview

Trigger:
- When chat message received

AI Agent:
- Chat Model: Google Gemini
- Memory: Simple Memory (stores conversation context)
- Tool: Wikipedia (for external knowledge lookup)

## 🧠 Features

- Context-aware responses using memory
- Tool-augmented generation (Wikipedia integration)
- Modular AI agent design
- Easy to extend with more tools

## 🚀 Use Cases

- Chatbots
- Knowledge assistants
- Automation workflows with AI
- Personal AI assistant

## 🛠️ Tech Stack

- n8n
- Google Gemini API
- Wikipedia Tool
- Simple Memory Node
